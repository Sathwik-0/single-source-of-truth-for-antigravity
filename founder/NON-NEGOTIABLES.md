# Chronicle Non-Negotiables

These rules override feature requests, deadlines, and convenience.

## 1. No Fake Buttons

Every button must execute real functionality.

Forbidden:

- Placeholder buttons
- "Coming Soon" disguised as functionality
- Fake success messages

Every action must map:

UI
↓
API
↓
Service
↓
Database
↓
Response

---

## 2. No Placeholder Integrations

An integration exists only when:

- OAuth works
- Data sync works
- Error handling works
- Reconnect works

Otherwise it does not exist.

---

## 3. Evidence Before AI

AI answers are not truths.

Every answer should contain:

- Sources
- Timeline
- Evidence
- Confidence

---

## 4. Ask Before Inventing

If requirements are unclear:

STOP

Ask founder.

Never invent product behavior.

---

## 5. Fast By Default

Latency is a feature.

Every screen should feel instant.

---

## 6. Memory First

Chronicle exists to preserve understanding.

Not documents.
Not files.
Not tickets.

Understanding.
