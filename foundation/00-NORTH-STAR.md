# North Star

## Mission

Reduce Time To Understanding.

Chronicle exists because engineering organizations lose understanding faster than they lose code.

Code survives.

Context disappears.

Decisions disappear.

Incidents disappear.

Chronicle reconstructs understanding.

---

## Success Metric

Time To Understanding

TTU measures how quickly an engineer can answer:

- Why did this happen?
- Who changed this?
- What caused this?
- What should we do next?

---

## Product Goal

Make engineers instinctively ask Chronicle questions.

Chronicle should feel like:

Perplexity + Linear + GitHub + Cursor

Fast.

Beautiful.

Evidence-first.

Keyboard-driven.

---

## Principles

Trust > Features

Evidence > AI

Memory > Retrieval

Understanding > Search

---

## Long-Term Goal

Become the memory layer for engineering organizations.
