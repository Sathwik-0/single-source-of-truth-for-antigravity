# Monetization Strategy

Chronicle's pricing plans align with the value delivered: team size, integration complexity, and repository memory scaling.

---

## 1. Free Tier (Solo & Hobbyists)
* **Price:** $0/month
* **Audience:** Individual developers, open-source maintainers, and hobbyists.
* **Features:**
  * 1 Public Repository index.
  * Basic search and vector retrieval.
  * Local-only memory index.
  * Single-user workspace.

---

## 2. Pro Tier (Professional Developers)
* **Price:** $29/month per user (Billed monthly)
* **Audience:** Freelancers, contract developers, and individual contributors in mid-sized firms.
* **Features:**
  * Unlimited public and private repositories.
  * Core Timeline and Memory engines.
  * Multi-repo cross-referencing.
  * Advanced context compression (handling codebases up to 1M lines).
  * Priority indexing queues.

---

## 3. Team Tier (High-Velocity Startups)
* **Price:** $299/month (Flat rate for up to 10 users, +$15/user/month additional)
* **Audience:** Startup teams (5–50 developers) needing shared organizational context.
* **Features:**
  * Complete Slack integration (channels, threads, user activity mapping).
  * Shared Timeline and Memory graphs.
  * Shared Decision Logs and automated Incident timelines.
  * Linear and Jira integration (when V2 launches).
  * Daily memory consolidation reports.

---

## 4. Enterprise Tier (Secure Corporations)
* **Price:** Custom Annual Pricing (Contract-based)
* **Audience:** Large technology organizations (50+ developers) with strict security and scale requirements.
* **Features:**
  * Single Sign-On (SSO) and Active Directory integrations.
  * Advanced Role-Based Access Control (RBAC).
  * Audit logs and compliance reporting (SOC2, GDPR).
  * On-Premise deployment or Single-Tenant Dedicated Cloud hosting.
  * Real-time sync engine with custom rate limit management.
  * Dedicated customer success engineering and custom SLA agreements.
